# Poultry-Disease-Classification

The Poultry Disease Detection System is a deep learning-based application designed to identify common poultry diseases using fecal image analysis. The system classifies four categories: Coccidiosis, Healthy, New Castle Disease, and Salmonella. A transfer learning approach is implemented using a pretrained convolutional neural network to ensure high accuracy and efficient feature extraction. Data preprocessing and augmentation techniques are applied to improve generalization and handle dataset imbalance.

The model achieves 98% test accuracy with strong Precision, Recall, and F1-scores of 0.98, demonstrating robust classification performance. ROC analysis shows excellent class separability with an AUC score close to 1.00. The system enables early disease detection, reducing mortality rates and economic losses in poultry farming. Overall, the project contributes to smart agriculture by providing an efficient, scalable, and reliable AI-based diagnostic solution.
